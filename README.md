# Телеграм-бот для отслеживания привычек

## Техническое задание для Телеграм-бота

**Описание проекта:** Telegram-бот “Трекер привычек”
**Название проекта:** Трекер привычек “HabiBot”

### Список ролей и функций

#### Пользователь

- Регистрация и настройка бота.
- Добавление новых привычек.
- Просмотр прогресса по привычкам.
- Получение напоминаний о привычках.
- Удаление или редактирование привычек.
- В привычке может редактироваться: частота повторения, название.

#### Бадди

- Опциональная роль для продвинутой версии бота. Позволяет делится с друзьями своим прогрессом по внедрению привычек и поддержкой друг друга.
- Подписка на ежедневный или еженедельный дайджест по привычкам выбранных пользователей
- Разрешение другим пользователям подписываться на свои дайджесты выполнения привычек.
- Комментирование дайджестов бадди.
- Управление своими подписками и разрешениями.

#### Администратор

- Опциональная роль для продвинутой версии бота.
- Управление списком пользователей.
- Модерация.
- Настройка глобальных параметров бота.
- Обновление функциональности и управление базой данных.

### Базовые функции

1. Регистрация пользователя и добавление привычек.
2. Напоминания о выполнении привычек.
3. Ведение статистики выполнения привычек.
4. Просмотр отчета за день/неделю/месяц.

#### Запросы к боту: примеры

##### Регистрация

- Пользователь: /start
- Бот: Привет! Я помогу тебе отслеживать привычки. Как тебя зовут?
- Пользователь: Лёша
- Бот: Отлично, Лёша! Давай начнем!

##### Добавление привычки

- Пользователь: /add
- Бот: Напиши привычку, которую ты хочешь отслеживать:
- Пользователь: Медитировать
- Бот: В какое время напоминать тебе об этой привычке? (например, 08:00)
- Пользователь: 08:00
- Бот: Готово! Я напомню тебе медитировать каждый день в 08:00.

#### Получение отчета

- Пользователь: /stats
- Бот: Твои привычки за неделю:
- Медитировать: 5/7 дней
- Упражнения: 3/7 дней`

##### Напоминание

- Бот: Напоминание: не забудь медитировать!
- Пользователь: Выполнено!
- Бот: Молодец! Я записал, что ты выполнил эту привычку.

### Идеи для развития

Далее, например, можно расширять отчёт, добавить список привычек для выбора с рекомендуемой предустановленной частой повторения, предлагать программы и рекомендации внедрения привычек в зависимости от цели пользователя, сделать откладывание привычек, более одного повторения привычки с одним названием в день (например медитировать утром, днем и перед сном), напоминания о приёме лекарств, питании и тому подобное. Отслеживать прогресс как для индивидуальных привычек, так и для групповых (если роль “Бадди” активирована).
